# 30均线趋势过滤器定投策略 — 参考文档

## 策略原理

### 什么是定投？

定期定额投资（Dollar-Cost Averaging, DCA）：无论市场涨跌，以固定周期（如每月）投入固定金额购买资产。优势是无需判断市场时机，自动摊薄持仓成本。

### 什么是趋势过滤器？

利用移动均线（MA）识别市场当前趋势方向：
- 收盘价 **高于** MA30 → 上升趋势（多头区间）
- 收盘价 **低于** MA30 → 下跌趋势（空头区间）

MA30 = 过去30个交易日收盘价的简单平均值。

---

## 四种策略对比

### 策略1：普通定投（基准）
```
每月第一交易日，固定投入 ¥1,000
不考虑均线信号
```
- 优点：操作最简单，完全自动化
- 缺点：无法利用低位机会加大投入

---

### 策略2：趋势加倍定投
```
收盘 >= MA30 → 投入 ¥1,000（正常）
收盘 <  MA30 → 投入 ¥2,000（加倍）
```
- 核心思路：低位多买，加大摊薄力度
- 适合：相信长期上涨、愿意在下跌时坚持加仓的投资者
- 注意：需要有额外现金储备

---

### 策略3：趋势暂停定投
```
收盘 >= MA30 → 投入 ¥1,000（正常）
收盘 <  MA30 → 暂停当月（不投）
```
- 核心思路：规避下跌趋势，只在上涨趋势中买入
- 适合：风险厌恶型，不想买在高位
- 风险：可能错过底部加仓机会，总投入金额大幅减少

---

### 策略4：趋势分级定投（推荐）
```
偏离度 = (收盘价 - MA30) / MA30

偏离度 >=  5%  → 投入 ¥500   (0.5倍，均线上方较多，偏贵)
偏离度 ≥   0%  → 投入 ¥1,000 (1倍，均线附近，正常)
偏离度 >= -5%  → 投入 ¥1,500 (1.5倍，略低于均线)
偏离度 >= -10% → 投入 ¥2,000 (2倍，低于均线较多)
偏离度 <  -10% → 投入 ¥3,000 (3倍，严重低于均线，抄底)
```
- 核心思路：价格离均线越远越便宜，买越多；越贵买越少
- 适合：追求更优平均成本、风险承受力中等的投资者
- 优势：比"加倍"更精细，比"暂停"更充分利用低位

## 关键指标解释

| 指标 | 说明 |
|------|------|
| 总投入 | 回测期间实际投入的资金总额 |
| 最终市值 | 以最后一个交易日收盘价计算的持仓价值 |
| 总收益率 | `(最终市值 - 总投入) / 总投入 × 100%` |
| 年化收益率 | `(最终市值/总投入)^(1/年数) - 1`，复利年化 |
| 最大回撤 | 组合市值从历史高点到最低谷的最大跌幅，衡量风险 |
| 定投次数 | 实际发生买入操作的月份数 |
| 暂停次数 | 因趋势过滤而暂停投入的月份数 |

## 使用建议

1. **标的选择**：适合宽基指数ETF（如沪深300、中证500、纳斯达克100等），不适合个股（波动过大）
2. **均线周期**：30日均线（约1.5个月）是中期趋势过滤的经典参数；也可尝试20日（短期）或60日（长期）
3. **基础金额**：建议占月收入的 10%-20%，确保在"加倍/3倍"月份有充足资金
4. **回测局限性**：
   - 历史回测不代表未来
   - 未考虑税费、手续费（ETF实际成本约0.01-0.1%）
   - 未考虑复权（如有分红）
5. **心理准备**：趋势加倍/分级策略在熊市中会持续加大投入，需要有"越跌越买"的心理承受力
